Netflix Dataset Analysis

This repository contains a Jupyter Notebook for the analysis and visualization of Netflixâ€™s open content dataset. The project enables users to explore, clean, and visualize Netflix data, uncovering patterns in categories, genres, geographic distribution, and content addition over time.

Features:

Loads Netflix data (CSV) and previews structure, columns, and sample entries.

Cleans data by handling missing values, dropping duplicates, and filling default fields for Country, Rating, and Duration.

Analyzes and visualizes content types (Movies vs TV Shows), top countries contributing content, genre distributions, and yearly addition of new titles.

Uses Python libraries (Pandas, Matplotlib, Seaborn) for data manipulation and plotting.

Step-by-step workflow, guiding from import to summary visualizations.

pandas

numpy

seaborn

Output:

Notebook creates:

Distribution plots for Movies vs TV Shows

Year-wise addition of content by category
